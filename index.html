Fase 2: Banco de Dados em Tempo Real

Para que todos os integrantes acessem e editem o repertório simultaneamente em tempo real na nuvem, recomendamos integrar a API do Supabase (gratuita).

Roteiro de Implementação Supabase:

  • Crie um projeto gratuito em supabase.com.
  • Crie as tabelas musicas e missas com a estrutura JSON gerada no exportador.
  • Importe o SDK oficial no HTML: <script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js"></script>.
  • Substitua os comandos do local storage do ministerio.js por chamadas de API do Supabase (ex: supabase.from('musicas').select()).
